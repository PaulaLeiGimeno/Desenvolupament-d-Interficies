CercaNoTrobada

Mostra un missatge d'error quan no troba cap registre en una cerca genèrica.

CercaNoTrobada
Parameters
props (Object) Propietats del component.
Name Description
props.missatgeError string Missatge d'error a mostrar.
props.textACercar string Text utilitzat en la cerca realitzada.
Example
<CercaNoTrobada
  missatgeError="No he trobat cap partit amb equips que continguen:"
  textACercar="Florida Universitària"
/>

InfoEquip

Mostra la informació d'un equip en forma de targeta. La informació es mostra verticalment, una baix de l'altra. Consta de la imatge de l'escut de l'equip de futbol, el nom de l'equip i l'any de fundació.

InfoEquip
Parameters
props (Object) Propietats del component.
Name Description
props.nom string Nom de l'equip del qual volem mostrar la informació .
Example
<InfoEquip nom="Valencia" />

InfoPartit

Mostra la informació d'un partit en forma de targeta. Un partit consta de dos equips (local i visitant), l'hora a la que es jugarà el partit, l'estadi on es jugarà el partit. Si no es posa estadi, es mostrarà l'estadi de l'equip local.

InfoPartit
Parameters
props (Object) Propietats del component.
Name Description
props.local string Nom de l'equip local (qui juga 'en casa').
props.visitant string Nom de l'equip visitant.
props.hora string Hora del partit.
props.estadi string? (default null) prop opcional. Si s'omet es mostra l'estadi de l'equip local
Example
<InfoPartit
  local="Valencia"
  visitant="Barcelona"
  hora="20:30"
  estadi="Estadi Municipal Toni Puchades (Sueca)"
/>
