### Repo base per a l'examen 1a av 25-26 ###
Aquest serà el repositori base per a l'examen de la 1a avaluació de DIN.
#### instruccions ####
1. Caldrà `Clonar` el repositori a una carpeta local vostra. Feu:
    Obrir un terminal (cmd) i anar a la carpeta on teniu els vostres projectes.
    Executar  ``` git clone https://github.com/Florida2DAM/2526ex1av.git ./examen2526 ```
2. Executar una a una les següents ordres:
   ```
   cd examen2526
   npm install
   code .
   npx expo start
    ```
3. Comproveu que us funciona, i mireu el codi de App.js i dels components situats en la carpeta ./components/.
4. Podeu consultar la documentació de cada component entrant en la carpeta ./docs amb un navegador, i clicant el fitxer index.html
5. Si teniu algun problema, envieu-me un correu.
